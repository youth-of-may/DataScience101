# Machine Learning Portfolio

This repository showcases a collection of **machine learning and data analysis projects** implemented in Python using Jupyter Notebooks. The projects demonstrate hands-on experience across **supervised learning, unsupervised learning, and time series forecasting**, using real-world datasets.

Each notebook is designed to highlight practical problem-solving, data preprocessing, model development, and evaluation techniques commonly used in industry.

---

### Projects Overview

### Lung Cancer Prediction
**File:** `LungCancerModelAnalysis.ipynb`  
- Built classification models to analyze lung cancer data  
- Performed data cleaning, feature selection, and model evaluation  
- Assessed model performance to support predictive healthcare analysis  

---

### Bank Loan Approval Prediction
**File:** `SupervisedLearning(BankLoanApproval).ipynb`  
- Developed supervised learning models to predict loan approval outcomes  
- Conducted exploratory data analysis (EDA) and feature engineering  
- Evaluated classification performance using standard metrics  

---

### Nike Stock Price Forecasting
**File:** `TimeSeriesForecasting(NikeStock).ipynb`  
- Analyzed historical stock price data  
- Applied time series forecasting techniques to predict future trends  
- Demonstrated handling of time-dependent financial data  

---

### Wine Dataset Clustering
**File:** `UnsupervisedLearning(WineDataset).ipynb`  
- Applied unsupervised learning techniques to unlabeled data  
- Used clustering algorithms to identify natural groupings  
- Interpreted cluster results to extract meaningful insights
